### 【Q】请解释类和对象？

类：具有共同属性或行为的对象的抽象。类是对象的模板。

对象：现实世界中的实体，比如：树，花。在计算机中，是指可标识的存储区域。

类和对象：类是对象的抽象，对象是类的实例。

比如：张三和李四都是人，都可以吃喝等行为，都有身高体重等属性。
