### 如何在网页中使用新的字体？

使用@font-face

```
@font-face {
    font-family: 'vicons';
    font-weight: normal;
    font-style: normal;
    src: url('../fonts/vicons/vicons.eot?1u5j7r');
    src: url('../fonts/vicons/vicons.eot?#iefix1u5j7r') format('embedded-opentype'), 
url('../fonts/vicons/vicons.woff?1u5j7r') format('woff'), 
url('../fonts/vicons/vicons.ttf?1u5j7r') format('truetype'), url('../fonts/vicons/vicons.svg?1u5j7r#vicons') format('svg');
}
```
